# Manuscript Numerical Experiments

This folder contains the **four validated numerical examples** presented in the CMAME paper
*"Mapped Sphere Method for Complex Geometry"*.

Each subfolder is self-contained and can be run independently.

---

## Example 1: Forward Poisson on 3D Ellipsoid / Star Domain

**Folder**: `example1_forward_poisson/`

| Script | Description |
|--------|-------------|
| `pinn_3d_ellipsoid_mapped_sphere.py` | Mapped-sphere PINN on a 3D ellipsoid |
| `run_poisson_star3d_mapped.py` | Mapped-sphere PINN on a 3D star domain |

**PDE**: -Delta u = f with manufactured solution u = sin(pi x) sin(pi y) sin(pi z)

---

## Example 2: Forward Poisson on Stanford Rabbit (Schwarz)

**Folder**: `example2_rabbit_poisson/`

| Script | Description |
|--------|-------------|
| `run_poisson_rabbit_atlas_schwarz.py` | Multi-chart PINN with multiplicative Schwarz |
| `run_poisson_rabbit_atlas_schwarz_fem.py` | FEM reference solve on the same atlas |
| `chart_fem_solver.py` | P1 FEM solver for individual charts |

**Prerequisites**: Trained SDF and atlas (see `core/train_sdf_rabbit.py` and `core/train_rabbit_atlas.py`)

**Best result**: rel-L2 = 2.21% (CompactChartNet, 61 Schwarz iterations)

## Reproducing Results

Each script can be run directly:

```bash
# Example 1: Ellipsoid Poisson
python manuscript_experiments/example1_forward_poisson/pinn_3d_ellipsoid_mapped_sphere.py

# Example 2: Rabbit Poisson (requires pre-trained SDF + atlas)
python manuscript_experiments/example2_rabbit_poisson/run_poisson_rabbit_atlas_schwarz.py

# Example 3: Torus inverse
python manuscript_experiments/example3_torus_inverse_original/run_torus_inverse_neohookean_atlas.py
```

Output data is saved to the `runs/` directory at the repository root.

## Figure Generation
